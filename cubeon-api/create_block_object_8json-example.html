create_block_object.json

This is an example how to create your own block in assets/objects/blocks/

{
"name": "example", //The name of block
"color": [255, 255, 255], //Color of block. Color is bounding in [0, 255] in rgb. Default is white color.
"texture": "example.png", //Name of texture, which placed in assets/textures/blocks
"box": [0.0, 0.0, 0.0, 1.0, 1.0, 1.0], //Set size of block. First three value is start offset and second three value is end offset. Limit of size [0.0-1.0]. Here is default value.
"active": false, //Here you can set block as active, what means, that block have script. Here is default value.
"transparency": false //Transparency param necessary for transparent textures. Here is default value.
"static": { //Set the static params of block. Which means params, what never changes for this type of block, constants.
"param1": 0,
"param2": "example"
}
"dynamic": { //Set the dynamic params of block. Which means params, individually changing for every block in world of this type
"param3": 1.0
"param4": "example"
}
}
